# COVID-19 India Tracker

**The Aim of the app is to give latest information regarding the COVID-19 cases all over India**

**Hope you guys will like this app ! Show some ❤️ and star the repo to support the project !**

# Features

* Daily Report
* State wise reports
* Graphical Stats
